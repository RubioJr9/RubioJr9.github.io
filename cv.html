1.1. Projects

In the course, CS 5610 Interactive Computer Graphics, I created a 3D Volume Render in C++ with OpenGL. I applied a color map, applied a threshold, stored the data in a 3D texture, and rendered using a GPU ray marching approach. I used this to visualize a CT scan of a carp fish.

In the course, CS 5635 Visualization for Scientfic Data, I worked in a group of 3 where we created a program to reconstruct photographs of heart slices as a 3D dataset. Since the images were extremely large, the program also trimmed empty space around the tissue and downscaled the resolution. We then used Paraview to visualize this with a transfer function so that we can isolate specific tissues in the data. We made the program generic so that my teammate’s lab could reuse this for future data samples.

In the course, EAE 3710 Traditional Game Development, I worked with a group of 6 to make a 3D twin-stick shooter game in Unity. I created most of the game logic, object movement, camera movement, and shaders. This was a good learning experience in linear algebra, especially writing target seeking bullets and remote controlled bullets. Our game qualified to be showcased at EAE day on campus.
